Suivi-Santé (Health Tracker)

Bienvenue sur **Suivi-Santé**, une application web simple permettant d'enregistrer et de suivre vos informations de santé au quotidien.

Fonctionnalités de l'application

L'application est divisée en plusieurs rubriques pour vous aider à mieux gérer votre santé :

- Profil santé (`profile.html`) : Renseignez et mettez à jour vos informations personnelles et médicales de base.
- Médicaments (`medication.html`) : Enregistrez vos traitements en cours, les dosages et les horaires de prise.
- Vaccins (`vaccines.html`) : Suivez vos vaccinations, enregistrez les dates et consultez votre historique vaccinal.
- Historique médical (`records.html`) : Gardez une trace de vos consultations, maladies, examens et autres événements importants liés à votre santé.
- Récapitulatif (`recap.html`) : Consultez une vue d'ensemble claire et organisée de toutes vos données de santé enregistrées.
- Conseils santé (`tips.html`) : Découvrez des recommandations pour adopter de bonnes habitudes et améliorer votre bien-être au quotidien.

Installation & Utilisation

1. Cloner ou télécharger le projet sur votre ordinateur.
2. Ouvrir le fichier principal: Double-cliquez sur `index.html` pour l'ouvrir dans votre navigateur web préféré (Chrome, Firefox, Safari, Edge, etc.).
3. Aucune installation de serveur ou de base de données complexe n'est requise. L'application fonctionne directement dans le navigateur grâce aux technologies web standards.

 Technologies utilisées

- HTML5 : Structure des pages web.
- CSS3: Mise en forme et style de l'application (`css/style.css`).
- JavaScript: Interactivité et logique front-end (`js/script.js`).


Ce projet a été conçu pour faciliter le suivi personnel de votre santé.
